Hexo 因主题文件夹存在 .git 而无法提交的解决方法

Posted on 2020-07-28 Edited on 2023-06-25 In Technology , JavaScript

问题原因

无法提交根本原因是next主题也是一个repo。

解决方法

1、剪切 themes/next/.git 文件夹到其他处

或者直接删除（视情况而定）

2、从暂存区删除该文件夹

3、使用 git status 查看状态

4、提交三连

5、移回 themes/next/.git 文件夹（未删除处理）
